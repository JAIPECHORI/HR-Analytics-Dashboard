# HR Analytics Dashboard

## Project Overview

This project analyzes employee attrition, workforce demographics, salary distribution, overtime impact, and job role trends using the IBM HR Employee Attrition dataset.

## Tools Used

* Power BI
* SQL
* Excel

## Key Insights

* Total Employees: 1470
* Attrition Rate: 16.12%
* Research & Development is the largest department.
* Employees with overtime show higher attrition.
* Lower salary bands have higher attrition rates.

## Dashboard Pages

### Page 1: Executive Overview

* Workforce by Department
* Workforce by Job Role
* Employee Attrition Overview
* Employee Demographics

### Page 2: Attrition Analysis

* Attrition by Department
* Attrition by Salary Band
* Attrition by Age Group
* Overtime vs Attrition
